PROYECTO TERMOFORMADORA

Proyecto 1: Termoformadora Automatizada

Este proyecto documenta la reconstrucción, puesta en marcha y análisis de viabilidad de una termoformadora industrial.

Pilares del Proyecto

Ingeniería Inversa y Reconstrucción

Se realizó la ingeniería inversa completa del equipo sin documentación original, permitiendo su reparación, modernización y optimización.

Automatización y Control

Se implementó un sistema de control centralizado mediante un PLC Kinco, programando la lógica para un ciclo de operación automático y fiable.

Análisis de Viabilidad

Se elaboró un modelo de negocio, investigando costos reales de mercado y proyectando la rentabilidad para demostrar su potencial comercial.

Subsistemas de la Máquina

Sistema Neumático

Cilindros de doble efecto (salidas Q0.2-Q0.4) ejecutan los movimientos mecánicos precisos del carro y el marco.

Sistema de Calentamiento

Resistencias (salidas Q2.0-Q2.5) llevan la lámina plástica a su temperatura de transición vítrea, volviéndola maleable.

Sistema de Vacío

La bomba de vacío (salida Q0.1) succiona el aire para forzar el moldeo del plástico.

Sistema de Enfriamiento

Un motoventilador (salida Q0.5) enfría la pieza para solidificarla rápidamente y completar el ciclo.

PROYECTO LÍNEA DE ENVASADO

Proyecto 2: Línea de Envasado Automatizada

Este proyecto cubre el diseño, la construcción y la automatización de una línea de envasado modular, utilizando una cinta transportadora como base.

Objetivos del Proyecto

  • Construir una cinta transportadora funcional como base del sistema.
  • Diseñar e integrar módulos para llenado, tapado y cerrado de botellas.
  • Desarrollar un sistema de control con PLC para automatizar la secuencia completa.

Análisis Funcional Animado

Desglose de la ingeniería de la línea de envasado y una simulación de su operación.

Módulos del Sistema

Módulo 1: Transporte

La cinta (salida Q1) avanza la botella. Cilindros de parada (Q3, Q5) la posicionan en cada estación.

Módulo 2: Llenado

Un cilindro (Q4) abre una tolva por un tiempo programado para dosificar el producto.

Módulo 3: Tapado y Cierre

Un dispensador posiciona la tapa (M) y un cilindro (Q6) baja un atornillador neumático (Q7) para el cierre hermético.
